# Sistema de Delivery 
Aplicação desenvolvida durante a Semana DevSuperior 2.0. O projeto consiste em um sistema de registro e entrega de pedidos.

### Ferramentas utilizadas:
- Java
- Spring Boot
- TypeScript
- ReactJs

### Projeto em produção:
https://snackdelyveri.netlify.app/
